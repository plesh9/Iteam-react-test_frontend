# 🏢 Job Search App

This project is a job search application built with **Next.js 15**, following a **custom FSD (Feature-Sliced Design) architecture**.  
It allows users to search for jobs using the **JSearch API**, save favorite jobs, and create a profile.

## 🚀 Technologies & Architecture

- **Next.js 15** – Modern React framework for both server-side and client-side rendering.
- **Custom FSD Architecture** – Modular structure with clear separation of responsibilities.
- **Vibe Library (Custom npm library)** – UI components and reusable elements.
- **Custom Utility for Caching & Query Updates** – Optimized API request handling.

## 🔥 Features

- **Public, Auth, and UnAuth Routes** – Well-structured routing system.
- **Job Search** – Users can search for jobs by title.
- **Job Details Page** – Each job has a dedicated page with detailed information.
- **Favorite Jobs** – Users can like jobs and save them to LocalStorage.
- **Profile Creation** – Users can create a profile stored in LocalStorage.
